# Terminos y condiciones del servicio

Este documento informa los terminos del servicio de la App de Discord RolThemesBot sobre la cual el unico que debe poder usarla es el mismo desarrollador.

# Politica de privacidad

La App de Discord RolThemesBot no recopila ni almacena ninguna información personal de los usuarios. No se recopilan datos de uso, preferencias o cualquier otra información identificable. La privacidad de los usuarios es una prioridad y no se comparte ninguna información con terceros.
